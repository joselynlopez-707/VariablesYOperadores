# VariablesYOperadores

## ¿Para què se utilizan los operadores lògicos en programaciòn?
Los operadores lógicos se utilizan para combinar condiciones y evaluar expresiones booleanas, permitiendo tomar decisiones en el programa.

## ¿Por qè es importante declarar correctamente el tipo de dato de una variable?
Es importante declarar correctamente el tipo de dato porque cada tipo ocupa diferente espacio en memoria y permite realizar operaciones específicas. Usar el tipo incorrecto puede generar errores o pérdida de información.

## Justificación Técnica

Clasificación de edad:
Se utilizó if-else porque se evalúan rangos de valores y no valores exactos.

Día de la semana:
Se utilizó switch porque se comparan valores exactos (1 al 7).

Verificación de acceso:
Se utilizó if-else porque se deben evaluar múltiples condiciones combinadas con operadores lógicos.
